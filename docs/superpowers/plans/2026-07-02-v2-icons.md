# v2-icons Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the radial menu's 4 text labels with hand-drawn `QtQuick.Shapes` icons (speaker, sun, chevrons, play/pause), matching the existing gold/dark highlight color-inversion.

**Architecture:** A new reusable `DialIcon.qml` component maps `DialController::iconNameAt(index)`'s existing id string to one of four hand-drawn line-art glyphs. `RadialMenu.qml`'s delegate swaps its `Text` label for a `DialIcon`, reusing the delegate's existing highlight check to pick the icon's stroke color. Pure QML, zero C++ changes.

**Tech Stack:** QML, `QtQuick.Shapes` (already available via the `Qml`/`Quick` Qt modules already linked — no new dependency).

## Global Constraints

- No C++ changes — `DialController::iconNameAt(index)` already returns the 4 needed id strings (`"audio-volume-high"`, `"display-brightness"`, `"input-mouse"`, `"media-playback-start"`); this plan only adds a QML consumer.
- No new build dependencies — icons are `QtQuick.Shapes` geometry, not bundled SVG/PNG assets, not a shader/effects module.
- Icons replace text ONLY in the radial menu's 4 outer boxes. The "ASUS Dial" center-of-menu logo (`CenterLogo.qml`) and the HUD confirmation (`Hud.qml`) stay text, unchanged.
- Icon color must invert exactly the way the box background/border already does: gold (`#C9A87C`) on the dark unselected boxes, dark (`#1a1a1a`) on the gold highlighted box.
- No automated tests for this plan — QML visuals aren't unit tested in this project (established in the v1 plan's Task 10 and the ASUS Dial branding pass); "the project builds" is the verification step, and the user visually confirms the result by running the app.

### Task 1: DialIcon component + wire into RadialMenu

**Files:**
- Create: `openwheel-gadget/qml/DialIcon.qml`
- Modify: `openwheel-gadget/qml/RadialMenu.qml`
- Modify: `openwheel-gadget/CMakeLists.txt` (add `qml/DialIcon.qml` to `qt_add_qml_module(...)`'s `QML_FILES` list)

**Interfaces:**
- Consumes: `DialController::iconNameAt(index)` (already implemented, Task 8 of the v1 plan) — returns one of `"audio-volume-high"`, `"display-brightness"`, `"input-mouse"`, `"media-playback-start"`. `DialController::highlightedIndex` (already implemented) — used by `RadialMenu.qml`'s existing highlight check.
- Produces: `DialIcon.qml` — a QML component with properties `iconId: string` and `color: color` (default `"#C9A87C"`), rendering the matching glyph. Not consumed by any later task (this plan has only one task).

- [ ] **Step 1: Write `DialIcon.qml`**

```qml
// openwheel-gadget/qml/DialIcon.qml
import QtQuick
import QtQuick.Shapes

Item {
    id: root
    property string iconId: ""
    property color color: "#C9A87C"

    // Volume: speaker body + cone + two soundwave arcs
    Item {
        visible: root.iconId === "audio-volume-high"
        anchors.fill: parent

        Shape {
            anchors.fill: parent
            ShapePath {
                strokeColor: root.color
                strokeWidth: 1.6
                fillColor: "transparent"
                capStyle: ShapePath.RoundCap
                joinStyle: ShapePath.RoundJoin
                startX: 4; startY: 11
                PathLine { x: 9; y: 11 }
                PathLine { x: 15; y: 6 }
                PathLine { x: 15; y: 22 }
                PathLine { x: 9; y: 17 }
                PathLine { x: 4; y: 17 }
                PathLine { x: 4; y: 11 }
            }
            ShapePath {
                strokeColor: root.color
                strokeWidth: 1.6
                fillColor: "transparent"
                capStyle: ShapePath.RoundCap
                startX: 20; startY: 10
                PathAngleArc { centerX: 18; centerY: 14; radiusX: 4; radiusY: 4; startAngle: -40; sweepAngle: 80 }
            }
            ShapePath {
                strokeColor: root.color
                strokeWidth: 1.6
                fillColor: "transparent"
                capStyle: ShapePath.RoundCap
                startX: 23; startY: 7
                PathAngleArc { centerX: 18; centerY: 14; radiusX: 7; radiusY: 7; startAngle: -40; sweepAngle: 80 }
            }
        }
    }

    // Brightness: circle + 8 radiating rays
    Item {
        visible: root.iconId === "display-brightness"
        anchors.fill: parent

        Shape {
            anchors.fill: parent
            ShapePath {
                strokeColor: root.color
                strokeWidth: 1.6
                fillColor: "transparent"
                startX: 19; startY: 14
                PathAngleArc { centerX: 14; centerY: 14; radiusX: 5; radiusY: 5; startAngle: 0; sweepAngle: 360 }
            }
        }

        Repeater {
            model: 8
            delegate: Item {
                required property int index
                anchors.fill: parent
                rotation: index * 45

                Rectangle {
                    anchors.horizontalCenter: parent.horizontalCenter
                    y: 1
                    width: 1.6
                    height: 4
                    radius: 0.8
                    color: root.color
                }
            }
        }
    }

    // Scroll: stacked up/down chevrons
    Item {
        visible: root.iconId === "input-mouse"
        anchors.fill: parent

        Shape {
            anchors.fill: parent
            ShapePath {
                strokeColor: root.color
                strokeWidth: 1.8
                fillColor: "transparent"
                capStyle: ShapePath.RoundCap
                joinStyle: ShapePath.RoundJoin
                startX: 9; startY: 14
                PathLine { x: 14; y: 9 }
                PathLine { x: 19; y: 14 }
            }
            ShapePath {
                strokeColor: root.color
                strokeWidth: 1.8
                fillColor: "transparent"
                capStyle: ShapePath.RoundCap
                joinStyle: ShapePath.RoundJoin
                startX: 9; startY: 15
                PathLine { x: 14; y: 20 }
                PathLine { x: 19; y: 15 }
            }
        }
    }

    // Media: play triangle + pause bars
    Item {
        visible: root.iconId === "media-playback-start"
        anchors.fill: parent

        Shape {
            anchors.fill: parent
            ShapePath {
                strokeColor: root.color
                strokeWidth: 1.6
                fillColor: "transparent"
                joinStyle: ShapePath.RoundJoin
                startX: 7; startY: 9
                PathLine { x: 7; y: 19 }
                PathLine { x: 16; y: 14 }
                PathLine { x: 7; y: 9 }
            }
        }
        Rectangle { x: 19; y: 9; width: 2.2; height: 10; radius: 1; color: root.color }
        Rectangle { x: 23; y: 9; width: 2.2; height: 10; radius: 1; color: root.color }
    }
}
```

- [ ] **Step 3: Replace the delegate's `Text` with `DialIcon`**

Find this block inside the delegate `Rectangle` in `openwheel-gadget/qml/RadialMenu.qml`:

```qml
            Text {
                anchors.centerIn: parent
                text: dialController.displayNameAt(index)
                color: index === dialController.highlightedIndex ? "#1a1a1a" : "#C9A87C"
                font.pixelSize: 10
                font.letterSpacing: 0.5
                wrapMode: Text.WordWrap
                width: parent.width - 8
                horizontalAlignment: Text.AlignHCenter
            }
```

Replace it with:

```qml
            DialIcon {
                anchors.centerIn: parent
                width: 28
                height: 28
                iconId: dialController.iconNameAt(index)
                color: index === dialController.highlightedIndex ? "#1a1a1a" : "#C9A87C"
            }
```

(If the exact current block differs slightly from the snippet above — e.g. different property order — replace whatever `Text` element is there with the `DialIcon` block above, preserving everything else in the delegate untouched.)
